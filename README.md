## Qt & QML Application Template

<img width="662" height="538" alt="image" src="https://github.com/user-attachments/assets/ed6aa23f-4594-4f4d-b3e2-2d7f274e192c" />

A minimal and modular template for building modern applications with C++ and QML using Qt 6.  
It provides a clean foundation for scalable and maintainable UI projects, suitable for both rapid prototyping and production environments.

---

## Requirements

| Dependency | Minimum Version | Notes |
|-------------|----------------|-------|
| C++ Compiler | C++23 | Any modern compiler (MSVC, Clang, GCC) |
| CMake | 3.28+ | Build configuration system |
| Conan | 2.x | Dependency manager |
